# Sparky-Web-API
A simple Java Wrapper for the Sparky Web API
#### Maven:
```xml
    <repositories>
        <repository>
            <id>Sparky</id>
            <url>https://nexus.sparky.ac/repository/Sparky/</url>
        </repository>
    </repositories>

    <dependencies>
        <dependency>
            <groupId>ac.sparky</groupId>
            <artifactId>SparkyWebAPI</artifactId>
            <version>1.2</version>
        </dependency>
    </dependencies>
```
