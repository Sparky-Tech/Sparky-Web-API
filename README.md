# Sparky-Web-API
A simple Java Wrapper for the Sparky Web API
#### Maven:
```xml
	<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>

	<dependency>
	    <groupId>com.github.Sparky-Tech</groupId>
	    <artifactId>Sparky-Web-API</artifactId>
	    <version>1.2</version>
	</dependency>
```
